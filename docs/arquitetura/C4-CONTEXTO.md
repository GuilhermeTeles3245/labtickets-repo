## Explicação

**Sistemas / Atores Principais:**
1. **Professor / Aluno (Solicitante):** Usa o sistema web para abrir chamados.
2. **Técnico:** Usa o sistema para ver a fila e resolver chamados.
3. **Coordenador:** Usa o sistema para extrair métricas de performance.
4. **Sistema LabTickets:** O sistema em si que gerencia a fila.
5. **E-mail Server (Futuro):** Sistema externo para notificação de mudanças de status.
