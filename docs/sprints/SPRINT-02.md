# Sprint 02 — Setup do Banco de Dados e Login

• **Sprint Goal:** Ter o banco de dados rodando localmente com a funcionalidade de Login e Controle de Sessão concluída.
• **Itens selecionados (Sprint Backlog):** 
  - Criar models.py / app.py com SQLAlchemy.
  - Criar as tabelas Usuario, Chamado e Comentario.
  - Criar página base.html e login.html.
  - Implementar rota de autenticação segura.
• **Daily (resumo):** 
  - Dia 1: Configuração do Flask-SQLAlchemy. Impedimento com erro de versão resolvido no mesmo dia.
  - Dia 2: Tabelas criadas.
  - Dia 3: Frontend do Login feito com Bootstrap.
  - Dia 4: Integração Frontend e Backend do Login.
  - Dia 5: Testes de segurança e controle de rotas por perfil.
• **Review:** Demonstramos os três tipos de usuários logando no sistema com sucesso. Cliente achou a interface limpa.
• **Retrospective:** 
  - Ação 1: Começamos a codar sem desenhar a tela antes, vamos fazer rabiscos para a próxima sprint.
